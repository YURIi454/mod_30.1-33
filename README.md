Учебный проект посвящён Django REST Framework

Создан новый Django-проект, подключены DRF в настройках.

Созданы модели:

1.Пользователь

2.Курс

3.Урок

Настроена связь моделей "курс" и "урок"

Описан CRUD для моделей курса и урока. 
Для модели "курс" использован  Viewsets, 
а для модели "урок"  Generic.
